Vídeo de apresentação: https://youtu.be/lkQf3hwq7PI

Descrição:
ULA LÁ! é um jogo de cartas educativo desenvolvido para auxiliar no ensino de conceitos fundamentais de Arquitetura e Organização de Computadores (AOC). Inspirado na mecânica do jogo Saboteur, o jogo gamifica o fluxo de dados dentro de um sistema computacional, tornando mais palpáveis conceitos abstratos como barramentos, hierarquia de memória e tratamento de interrupções, além de claro, trazer diversão!
